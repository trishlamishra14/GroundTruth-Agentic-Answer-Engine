How it works

A small agent — not a one-shot pipeline. It grades its own retrieval, retries weak queries, and refuses when the answer isn't in the docs.

01

Retrieve

Hybrid search — vector similarity plus keyword matching — fused with Reciprocal Rank Fusion.

pgvector + BM25 + RRF
02

Re-rank

A cross-encoder reads the question and each passage together to score true relevance.

Cohere Rerank
03

Grade & loop

Strong enough? Answer. Too weak? The agent rewrites the query and retries.

LangGraph agent
04

Answer or abstain

Writes a cited answer from the sources — or says "not in the docs" rather than guess.

Cohere Command

Grounded

Every claim is backed by a retrieved passage, cited inline as [1], [2] — traceable, not invented.

Abstains

If the docs don't support an answer, it refuses — the single biggest defense against hallucination.

Evaluated

A gold benchmark scores recall, faithfulness, and abstention in CI — quality is measured, not assumed.

Why it matters

A confident wrong answer is worse than no answer — it erodes trust and creates cleanup work. GroundTruth refuses to guess, shows its sources, and proves its quality with a benchmark.

Docs & knowledge Q&A

Answer questions over engineering docs, SOPs, and policies — with the exact source passages attached, so answers are verifiable.

Why I built it

I build document-AI systems for a US-healthcare product, where a wrong answer isn't an option. I built GroundTruth to take that discipline — grounding, abstention, and real evaluation — and show it end to end as a clean, deployable service: an agent that retrieves, reasons, cites, and refuses, with a benchmark that runs in CI. Most RAG demos are a notebook; I wanted to build the version you'd actually put in production.
